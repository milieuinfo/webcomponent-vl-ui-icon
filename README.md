# <vl-icon> Web Component
De["icon" component van Webuniversum Vlaanderen](https://overheid.vlaanderen.be/webuniversum/v3/documentation/components/vl-ui-icon)als Web Component


## Installation
```
npm install --save vl-ui-icon
```

## Demo
```
npm run demo
```
